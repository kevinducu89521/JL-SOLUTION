# JL-SOLUTION 🚗

Full-stack website for a multi-service business offering **DMV paperwork, insurance, and notary services** in Jackson Heights, Queens, New York.

> Real client project — designed and developed from scratch.

## 🌐 Live Demo

🔗 **https://jlsolution.site** *(si la URL real es otra, reemplázala)*

## ✨ Features

- Fully responsive design for mobile, tablet, and desktop
- Smooth UI animations powered by Framer Motion
- Contact form with validation (React Hook Form + Zod) that sends emails through a Node.js + Nodemailer backend
- Accessible, modern interface built with Tailwind CSS and shadcn/ui (Radix UI) components

## 🛠️ Tech Stack

**Frontend**
- React 19 + TypeScript
- Vite
- Tailwind CSS & shadcn/ui (Radix UI)
- Framer Motion

**Backend**
- Node.js + Express
- Nodemailer

## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/kevinducu89521/JL-SOLUTION.git
cd JL-SOLUTION

# Install dependencies
npm install

# Run the development server
npm run dev
```

To build for production (frontend + backend):

```bash
npm run build
npm run build:backend
npm start
```
